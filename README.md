# 🎯 Number Guessing Game (Python)

A simple and interactive **Number Guessing Game** built using Python. The player selects a custom number range, optionally sets a maximum number of guesses, and tries to guess the randomly generated number.

## 🎮 How to Play

1. Enter the minimum number.
2. Enter the maximum number.
3. Choose whether you want to set a limit on the number of guesses.
4. If you choose **Yes**, enter the maximum number of guesses.
5. Start guessing the randomly generated number.
6. The game provides hints:

   * 📉 Too low!
   * 📈 Too high!
7. Keep guessing until you win or reach the guess limit.
